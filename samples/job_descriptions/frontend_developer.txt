Frontend Developer

Location: San Francisco, CA (Hybrid)
Type: Full-time

Job Description:

- Develop responsive web applications using modern JavaScript frameworks
- Implement pixel-perfect designs from Figma mockups

- Ensure cross-browser compatibility and accessibility standards

- Write clean, maintainable, and well-documented code
- Participate in code reviews and testing processes

- Familiarity with build tools (Webpack, Vite, Parcel)

- Knowledge of browser developer tools and debugging techniques

Preferred Skills:

- Knowledge of state management libraries (Redux, Vuex, MobX)

- Experience with CSS-in-JS libraries (Styled Components, Emotion)
- Understanding of web performance optimization techniques
- Knowledge of accessibility standards (WCAG)
- Experience with design systems and component libraries
- Familiarity with GraphQL
